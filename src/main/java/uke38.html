Oppgave 1: Areal av en sirkel

Lag et JavaScript-program som beregner arealet av en sirkel. Be brukeren om å skrive inn radiusen til sirkelen ved hjelp av prompt(). Bruk en variabel til å lagre radiusen og en annen variabel til å lagre resultatet av beregningen. Vis deretter resultatet ved hjelp av console.log(). Formelen for å beregne arealet av en sirkel er: A = π * r^2, der "A" er arealet og "r" er radiusen.


        
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>Beregn Sirkelens Areal</title>
</head>

<body>
    <script>
        // Be brukeren om å skrive inn radiusen
        var radius = prompt("Skriv inn radiusen til sirkelen:");

        // Beregn arealet av sirkelen
        var pi = Math.PI;
        var areal = pi * Math.pow(radius, 2);

        // Vis resultatet
        console.log("Arealet av sirkelen med radius " + radius + " er " + areal);
    </script>
</body>

</html>

Oppgave 2: Temperatur konvertering

Lag et JavaScript-program som konverterer temperatur fra Celsius til Fahrenheit. Be brukeren om å skrive inn temperaturen i Celsius ved hjelp av prompt(). Bruk en variabel til å lagre temperaturen i Celsius og en annen variabel til å lagre resultatet av konverteringen til Fahrenheit. Vis deretter resultatet ved hjelp av console.log(). Formelen for å konvertere fra Celsius til Fahrenheit er: F = (C * 9/5) + 32, der "F" er temperaturen i Fahrenheit og "C" er temperaturen i Celsius.


        
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>Temperaturkonvertering</title>
</head>

<body>
    <script>
        // Funksjon for å konvertere fra Celsius til Fahrenheit
        function celsiusTilFahrenheit(celsius) {
            return (celsius * 9 / 5) + 32;
        }

        // Funksjon for å konvertere fra Fahrenheit til Celsius
        function fahrenheitTilCelsius(fahrenheit) {
            return (fahrenheit - 32) * 5 / 9;
        }

        // Temperatur i Celsius
        var temperaturCelsius = 25;

        // Konverter til Fahrenheit
        var temperaturFahrenheit = celsiusTilFahrenheit(temperaturCelsius);

        // Vis resultatene
        console.log(temperaturCelsius + " grader Celsius tilsvarer " + temperaturFahrenheit + " grader Fahrenheit.");

        // Temperatur i Fahrenheit
        var temperaturFahrenheit2 = 77;

        // Konverter til Celsius
        var temperaturCelsius2 = fahrenheitTilCelsius(temperaturFahrenheit2);

        // Vis resultatene
        console.log(temperaturFahrenheit2 + " grader Fahrenheit tilsvarer " + temperaturCelsius2 + " grader Celsius.");
    </script>
</body>

</html>

Oppgave 3: Variabler og utdata i JavaSqript

Lag en HTML-fil som inneholder JavaScript-kode for å presentere informasjon om en person.

Bruk følgende variabler:

Navn med verdien "Line Jensen".

Alder med verdien 23.

Myndig med verdien true.

Høyde med verdien 1.80.

Deretter, basert på disse variablene, skal du bygge en streng som inneholder følgende informasjon: "Line Jensen er 23 år gammel og er myndig og har en høyde på 1.80." Dette skal vises på følgende måter: En pop-up dialogboks ved hjelp av alert(). Skriving til dokumentet ved hjelp av document.write(). Utlogging til konsollen ved hjelp av console.log(). Du må bruke betinget logikk for å sjekke om personen er myndig eller ikke, og inkludere denne informasjonen i teksten. Lag HTML-filen og JavaScript-koden som oppfyller kravene i oppgaveteksten.


        
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>JavaScript 1</title>
    <script type="text/javascript">
        const navn = "Line Jensen";
        const alder = 23;
        const myndig = true;
        const hoyde = 1.80;
        let ut = navn + " " + alder;
        if (myndig) {
            ut += " er myndig ";
        } else {
            ut += " er ikke myndig ";
        }
        ut += " og har en høyde på " + hoyde;
        alert(ut);
        document.write(ut);
        console.log(ut);
    </script>
</head>

<body>

</body>

</html>

Oppgave 4: Flere funksjonaliteter

Lag en HTML-fil som inneholder JavaScript-kode for å håndtere ulike hendelser. Bruk følgende funksjoner:

(1) klikk(innData): Denne funksjonen skal kalles når brukeren endrer verdien i en tekstboks. Funksjonen skal ta inn dataen som er skrevet inn og vise en melding med denne informasjonen ved hjelp av alert().

(2) klikkKnapp(): Denne funksjonen skal kalles når brukeren klikker på en knapp. Den skal vise en melding med teksten "Knappen er klikket!" ved hjelp av alert().

(3) klikkDiv(): Denne funksjonen skal kalles når brukeren klikker på en div. Den skal vise to meldinger: "Div er klikket!" ved hjelp av alert(), og "Etter alert inne i kikkDiv!" ved hjelp av console.log().

(4) endreFarge(): Denne funksjonen skal kalles når brukeren beveger musepekeren over en annen div. Den skal endre tekstens farge til blå når musepekeren er over div-elementet, og tilbake til svart når musepekeren beveger seg bort. Dette skal gjøres ved å endre CSS-stilen til div-elementet.

Bruk HTML- og JavaScript-koden gitt i oppgaveteksten som utgangspunkt, og legg til de nødvendige hendelsesbehandlerne for å oppnå ønsket funksjonalitet. Test løsningen for å sikre at hendelsene behandles riktig.


        
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>JavaScript 1</title>
    <script type="text/javascript">
        function innKlikk(innData) {
            let utData = "Du har skrevet inn følgende data: " + innData;
            alert(utData);
        }

        function klikkKnapp() {
            alert("Knappen er kikket!");
        }

        function klikkDiv() {
            alert("Div er kikket!");
            console.log("Etter alert inne i kikkDiv!");
        }
    </script>
</head>

<body>
    <input type="text" onchange="innKlikk(this.value);" /><br />

    <input type="button" value="Trykk meg" onclick="klikkKnapp();" />

    <div onclick="klikkDiv();">Trykk meg</div>

    <div onmouseover="style.color='blue';" onmouseout="style.color='black';">
        Rør ved meg!
    </div>
</body>

</html>
