Oppgave 1: API for vitenskapelig fakta

Lag en nettside med en p-tagg som viser en initiell tekst. Ved siden av p-taggen, legg til en knapp. Når knappen klikkes, skal teksten i p-taggen endres til en tilfeldig vitenskapelig faktum hentet fra følgende API: https://uselessfacts.jsph.pl/random.json?language=en. Bruk id for å identifisere både p-taggen og knappen. Implementer JavaScript-koden som bruker fetch for å hente tilfeldig vitenskapelig informasjon fra API-et og oppdatere p-taggen med den nye teksten.

                
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>Vitenskapelige Fakta</title>
    <script type="application/javascript">
        function hentVitenskapeligFakta() {
            const tekstTagg = document.getElementById("vitenskapeligFakta");
            fetch('https://uselessfacts.jsph.pl/random.json?language=en')
                .then(response => response.json())
                .then(data => {
                    tekstTagg.textContent = data.text;
                })
                .catch(error => {
                    console.error('Feil ved henting av data: ', error);
                });
        }
    </script>
</head>

<body>
    <p id="vitenskapeligFakta">Klikk på knappen for å få et vitenskapelig faktum.</p>
    <button onclick="hentVitenskapeligFakta()">Hent Faktum</button>
</body>

</html>
