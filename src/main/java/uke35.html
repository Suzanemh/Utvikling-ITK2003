Oppgave 1: Enkel kalkulator

Skriv et Python-program som tar inn to tall fra brukeren og utfører en av de fire grunnleggende regneoperasjonene (addisjon, subtraksjon, multiplikasjon eller divisjon) på tallene. Programmet skal deretter skrive ut resultatet.

                
# -*- coding: utf-8 -*-
def main():
    num1=float(input("Skriv inn det første tallet: "))
    operator = input("Skriv inn operatoren(+ - * /): ")
    num2=float(input("Skriv inn det andre tallet: "))

    if operator == "+":
        result = num1 + num2
    elif operator == "-":
        result = num1 - num2
    elif operator == "*":
        result = num1 * num2
    elif operator == "/":
        result = num1 / num2
    else:
        print("ugyldig operator")
        return
    print("Resultatet er: " , result)

main()

Samme oppgave på nytt, men denne gangen med språket Java.

                
        import java.util.Scanner;

            public class kalkulator {
                public static void main(String[] args) {
                    Scanner scanner = new Scanner(System.in);

                    System.out.print("Skriv inn det første tallet: ");
                    double num1 = scanner.nextDouble();

                    System.out.print("Skriv inn operatøren (+, -, *, /): ");
                    char operator = scanner.next().charAt(0);

                    System.out.print("Skriv inn det andre tallet: ");
                    double num2 = scanner.nextDouble();

                    double result = 0;

                    switch (operator) {
                        case '+':
                            result = num1 + num2;
                            break;
                        case '-':
                            result = num1 - num2;
                            break;
                        case '*':
                            result = num1 * num2;
                            break;
                        case '/':
                            result = num1 / num2;
                            break;
                        default:
                            System.out.println("Ugyldig operatør!");
                    }

                    System.out.println("Resultatet er: " + result);
                }
            }

Oppgave 2: Funksjonell melding

Skriv en funksjon visMelding som tar imot navnet ditt som argument og viser en melding på følgende format: "Hei, [Navnet ditt]! Velkommen til HTML og JavaScript-introduksjon."


        
            <!DOCTYPE html>
            <html lang="en">

            <head>
                <meta charset="UTF-8">
                <title>Title</title>
                <script type="application/javascript">
                    function visMelding(navn) {
                        document.write("Hei, " + navn + "!
                        Velkommen til HTML og JavaScript-introduksjon.");
                    }
                </script>
            </head>

            <body>
                <button onclick="visMelding('DittNavn')">Trykk meg</button>
            </body>

            </html>

Oppgave 3: Flere meldinger

Endre koden slik at når brukeren trykker på knappen, blir det vist to meldinger. Den første meldingen skal være "Velkommen!" og den andre meldingen skal være den nåværende tiden i formatet "Klokken er [Time:NåværendeMinutt]".


        
             <!DOCTYPE html>
            <html lang="en">

            <head>
                <meta charset="UTF-8">
                <title>Title</title>
                <script type="application/javascript">
                    function visVelkomst() {
                        document.write("Velkommen!
");
                        var nå = new Date();
                        var time = nå.getHours();
                        var minutt = nå.getMinutes();
                        document.write("Klokken er " + time + ":" + minutt);
                    }
                </script>
            </head>

            <body>
                <button onclick="visVelkomst()">Trykk meg</button>
            </body>

            </html>

Oppgave 4: Tallmultiplikasjon

Legg til et input-felt i HTML-koden der brukeren kan skrive inn et tall. Endre JavaScript-funksjonen slik at den multipliserer tallet brukeren skrev inn med 5 og viser resultatet i et popup-vindu når knappen trykkes.


        
             <!DOCTYPE html>
            <html lang="en">

            <head>
                <meta charset="UTF-8">
                <title>Title</title>
                <script type="application/javascript">
                    function multipliserMedFem() {
                        var inputTall = prompt("Skriv inn et tall:");
                        var resultat = parseFloat(inputTall) * 5;
                        alert("Resultatet er: " + resultat);
                    }
                </script>
            </head>

            <body>
                <button onclick="multipliserMedFem()">Trykk meg</button>
            </body>

            </html>
