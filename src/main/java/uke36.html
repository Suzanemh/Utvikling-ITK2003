Oppgave 1: Enkel velkomstmelding

Skriv en kode slik at den viser en enkel velkomstmelding når knappen trykkes.

                
<!DOCTYPE html>
    <html lang="en">

    <head>
        <meta charset="UTF-8">
        <title>Title</title>
        <script type="application/javascript">
            function visVelkomst() {
                document.write("Velkommen til vår nettside!");
            }
        </script>
    </head>

    <body>
        <button onclick="visVelkomst()">Trykk meg</button>
    </body>

    </html>

Oppgave 2: Endre bakgrunnsfarge

Legg til en funksjon i JavaScript som endrer bakgrunnsfargen til nettsiden når knappen trykkes. Du kan for eksempel bruke forskjellige fargenavn som "red", "blue", "green", osv.

                
        <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <script type="application/javascript">
        function endreBakgrunnsfarge(farge) {
            document.body.style.backgroundColor = farge;
        }
    </script>
</head>

<body>
    <button onclick="endreBakgrunnsfarge('blue')">Blå bakgrunn</button>
    <button onclick="endreBakgrunnsfarge('red')">Rød bakgrunn</button>
</body>

</html>

Oppgave 2: Funksjonell melding

Skriv en funksjon visMelding som tar imot navnet ditt som argument og viser en melding på følgende format: "Hei, [Navnet ditt]! Velkommen til HTML og JavaScript-introduksjon."


        
            <!DOCTYPE html>
            <html lang="en">

            <head>
                <meta charset="UTF-8">
                <title>Title</title>
                <script type="application/javascript">
                    function visMelding(navn) {
                        document.write("Hei, " + navn + "!
                        Velkommen til HTML og JavaScript-introduksjon.");
                    }
                </script>
            </head>

            <body>
                <button onclick="visMelding('DittNavn')">Trykk meg</button>
            </body>

            </html>

Oppgave 3: Bytte bilder

Lag en HTML-side med to bilder og en knapp. Lag deretter en JavaScript-funksjon som bytter bildene hver gang knappen trykkes. Start med det første bildet synlig, og når knappen trykkes første gang, skal det første bildet skjules og det andre bildet vises. Trykk deretter på knappen igjen for å bytte tilbake.


        
            <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>Bytte Bilder</title>
    <script type="application/javascript">
        var visBilde1 = true;

        function byttBilder() {
            var bilde1 = document.getElementById("bilde1");
            var bilde2 = document.getElementById("bilde2");

            if (visBilde1) {
                bilde1.style.display = "none";
                bilde2.style.display = "block";
            } else {
                bilde1.style.display = "block";
                bilde2.style.display = "none";
            }

            visBilde1 = !visBilde1;
        }
    </script>
</head>

<body>
    <img id="bilde1" src="bilde1.jpg" alt="Bilde 1">
    <img id="bilde2" src="bilde2.jpg" alt="Bilde 2" style="display: none;">
    <button onclick="byttBilder()">Bytt Bilder</button>
</body>

</html>

Oppgave 4: Generere tilfeldige tall

Lag en JavaScript-funksjon som genererer og viser et tilfeldig heltall mellom 1 og 100 når en knapp trykkes. Bruk Math.random()-funksjonen til å generere tallet.


        
             <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>Tilfeldig Tall Generator</title>
    <script type="application/javascript">
        function genererTilfeldigTall() {
            var tilfeldigTall = Math.floor(Math.random() * 100) + 1;
            alert("Det tilfeldige tallet er: " + tilfeldigTall);
        }
    </script>
</head>

<body>
    <button onclick="genererTilfeldigTall()">Generer Tilfeldig Tall</button>
</body>

</html>
