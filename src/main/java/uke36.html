Oppgave 1: Enkel velkomstmelding

Skriv en kode slik at den viser en enkel velkomstmelding når knappen trykkes.

                
<!DOCTYPE html>
    <html lang="en">

    <head>
        <meta charset="UTF-8">
        <title>Title</title>
        <script type="application/javascript">
            function visVelkomst() {
                document.write("Velkommen til vår nettside!");
            }
        </script>
    </head>

    <body>
        <button onclick="visVelkomst()">Trykk meg</button>
    </body>

    </html>

Oppgave 2: Endre bakgrunnsfarge

Legg til en funksjon i JavaScript som endrer bakgrunnsfargen til nettsiden når knappen trykkes. Du kan for eksempel bruke forskjellige fargenavn som "red", "blue", "green", osv.

                
        <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <script type="application/javascript">
        function endreBakgrunnsfarge(farge) {
            document.body.style.backgroundColor = farge;
        }
    </script>
</head>

<body>
    <button onclick="endreBakgrunnsfarge('blue')">Blå bakgrunn</button>
    <button onclick="endreBakgrunnsfarge('red')">Rød bakgrunn</button>
</body>

</html>

Oppgave 3: Bytte bilder

Lag en HTML-side med to bilder og en knapp. Lag deretter en JavaScript-funksjon som bytter bildene hver gang knappen trykkes. Start med det første bildet synlig, og når knappen trykkes første gang, skal det første bildet skjules og det andre bildet vises. Trykk deretter på knappen igjen for å bytte tilbake.


        
            <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>Bytte Bilder</title>
    <script type="application/javascript">
        var visBilde1 = true;

        function byttBilder() {
            var bilde1 = document.getElementById("bilde1");
            var bilde2 = document.getElementById("bilde2");

            if (visBilde1) {
                bilde1.style.display = "none";
                bilde2.style.display = "block";
            } else {
                bilde1.style.display = "block";
                bilde2.style.display = "none";
            }

            visBilde1 = !visBilde1;
        }
    </script>
</head>

<body>
    <img id="bilde1" src="bilde1.jpg" alt="Bilde 1">
    <img id="bilde2" src="bilde2.jpg" alt="Bilde 2" style="display: none;">
    <button onclick="byttBilder()">Bytt Bilder</button>
</body>

</html>

Oppgave 4: Generere tilfeldige tall

Lag en JavaScript-funksjon som genererer og viser et tilfeldig heltall mellom 1 og 100 når en knapp trykkes. Bruk Math.random()-funksjonen til å generere tallet.


        
             <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>Tilfeldig Tall Generator</title>
    <script type="application/javascript">
        function genererTilfeldigTall() {
            var tilfeldigTall = Math.floor(Math.random() * 100) + 1;
            alert("Det tilfeldige tallet er: " + tilfeldigTall);
        }
    </script>
</head>

<body>
    <button onclick="genererTilfeldigTall()">Generer Tilfeldig Tall</button>
</body>

</html>

Oppgave 5: Fargeendring med knapp

Lag en nettside som inneholder en firkantig div med en knapp ved siden av den. Når brukeren klikker på knappen, skal bakgrunnsfargen til div endre seg til en tilfeldig farge. Bruk id for å identifisere både div og knappen. Funksjonen skal hentes ved hjelp av getElementById.


        
<!DOCTYPE html>
<html>
<head>
    <title>Fargeendring med knapp</title>
</head>
<body>
    <div id="firkant" style="width: 100px; height: 100px; background-color: red;"></div>
    <button id="endreFargeKnapp">Endre Farge</button>

    <script>
        function endreFarge() {
            var firkant = document.getElementById("firkant");
            var farger = ["red", "blue", "green", "orange", "purple"];
            var tilfeldigFarge = farger[Math.floor(Math.random() * farger.length)];
            firkant.style.backgroundColor = tilfeldigFarge;
        }

        var endreFargeKnapp = document.getElementById("endreFargeKnapp");
        endreFargeKnapp.addEventListener("click", endreFarge);
    </script>
</body>
</html>

Oppgave 6: Teller med knapper

Lag en nettside med en div som viser en teller som starter på 0. Legg til to knapper ved siden av telleren: én for å øke telleren med 1, og én for å redusere den med 1. Bruk id for å identifisere div og begge knappene. Implementer funksjoner ved hjelp av getElementById for å oppdatere telleren når knappene klikkes.


        
<!DOCTYPE html>
<html>
<head>
    <title>Teller med knapper</title>
</head>
<body>
    <div id="teller">0</div>
    <button id="okningKnapp">Øk</button>
    <button id="reduksjonKnapp">Reduser</button>

    <script>
        var teller = 0;
        var tellerElement = document.getElementById("teller");

        function okTeller() {
            teller++;
            tellerElement.textContent = teller;
        }

        function reduserTeller() {
            teller--;
            tellerElement.textContent = teller;
        }

        var okningKnapp = document.getElementById("okningKnapp");
        var reduksjonKnapp = document.getElementById("reduksjonKnapp");

        okningKnapp.addEventListener("click", okTeller);
        reduksjonKnapp.addEventListener("click", reduserTeller);
    </script>
</body>
</html>

Oppgave 7: Tekstendring med knapp

Lag en nettside med en p-tagg som viser en initiell tekst. Ved siden av p-taggen, legg til en knapp. Når knappen klikkes, skal teksten i p-taggen endres til en tilfeldig generert tekststreng fra en offentlig tekstgenerator-API. Bruk id for å identifisere både p-taggen og knappen. Implementer JavaScript-koden som bruker fetch for å hente tilfeldig tekst fra API-et og oppdatere p-taggen med den nye teksten. Husk å deaktivere knappen mens teksten hentes, og aktiver den igjen etterpå for å forhindre flere forespørsler til API-et samtidig.


        
<!DOCTYPE html>
<html>
<head>
    <title>Tekstendring med knapp (API)</title>
</head>
<body>
    <p id="tekstParagraf">Klikk på knappen for å generere tilfeldig tekst.</p>
    <button id="endreTekstKnapp">Endre Tekst</button>

    <script>
        function hentTilfeldigTekst() {
            var tekstParagraf = document.getElementById("tekstParagraf");
            var endreTekstKnapp = document.getElementById("endreTekstKnapp");

            // Deaktiver knappen mens du henter tekst fra API-et.
            endreTekstKnapp.disabled = true;

            // Hent tekst fra Lorem Ipsum API.
            fetch('https://baconipsum.com/api/?type=meat-and-filler')
                .then(response => response.json())
                .then(data => {
                    // Bruk den første tekststrengen fra API-responsen.
                    tekstParagraf.textContent = data[0];
                })
                .catch(error => {
                    console.error('Feil ved henting av tekst:', error);
                })
                .finally(() => {
                    // Aktiver knappen igjen etter at teksten er hentet.
                    endreTekstKnapp.disabled = false;
                });
        }

        var endreTekstKnapp = document.getElementById("endreTekstKnapp");
        endreTekstKnapp.addEventListener("click", hentTilfeldigTekst);
    </script>
</body>
</html>
