Oppstartsuke med elever. Denne timen gikk til å bli kjent med elevene, deretter gi de en oppgave som går ut på å lage e enkel kalkulator.

Enkel kalkulator

Skriv et Java-program som tar inn to tall fra brukeren og utfører en av de fire grunnleggende regneoperasjonene (addisjon, subtraksjon, multiplikasjon eller divisjon) på tallene. Programmet skal deretter skrive ut resultatet.

                
                    # -*- coding: utf-8 -*-

def main():
    num1 = float(input("Skriv inn det første tallet: "))
    operator = input("Skriv inn operatøren (+, -, *, /): ")
    num2 = float(input("Skriv inn det andre tallet: "))

    if operator == "+":
        result = num1 + num2
    elif operator == "-":
        result = num1 - num2
    elif operator == "*":
        result = num1 * num2
    elif operator == "/":
        result = num1 / num2
    else:
        print("Ugyldig operatør!")
        return

    print("Resultatet er:", result)

if __name__ == "__main__":
    main()
